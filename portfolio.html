Project Name: Halifax Transit Tracker

Description: Developed a real-time bus tracking application for Halifax Transit using JavaScript, Leaflet.js, HTML, and CSS. The app provides a user-friendly interface with a detailed map to visualize active bus routes and stops across the city. Designed for seamless navigation and enhanced transit planning, the tool ensures efficiency and convenience for daily commuters.

Project Name: 3-in-a-Row Game

Description: Designed and developed an interactive "3-in-a-Row" game using pure JavaScript and CSS. The game features a visually appealing grid layout where players can strategically align three blocks in a row to win. Includes functionality to check the puzzle, submit results, and reset the game for a new challenge. This project highlights dynamic DOM manipulation and CSS styling techniques for a responsive and engaging user experience.

Project Name: Cricket Player Stats Application

Description: Built a dynamic Player Stats application using React for the front-end, MongoDB for data storage, and Docker for containerized deployment. The app displays detailed profiles of players, including nicknames, roles, batting styles, and bowling styles, with an engaging card-based layout. This project highlights skills in full-stack development, database integration, and containerization for scalable and efficient deployment.
